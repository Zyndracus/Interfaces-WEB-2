Temas do Curso






Tipos de variáveis


Imagine que você está organizando uma caixa de brinquedos. Cada caixa tem um nome, como "caixa de bonecas" ou "caixa de carrinhos". Esses nomes são como variáveis em programação. Eles são contêineres onde você pode guardar diferentes coisas.

Operadores e estruturas de repetição


Os operadores em programação são como os sinais matemáticos que você usa em contas.


Às vezes, você precisa fazer a mesma coisa várias vezes. As estruturas de repetição em programação ajudam você a fazer isso de forma mais fácil e rápida.


Aprenda conosco clicando na imagem acima!



Lincar JavaScript e HTML


"Lincar" o JavaScript ao HTML significa conectar essas duas partes: o código JavaScript que dá vida à sua página e o código HTML que define sua estrutura. É como ligar a eletricidade para fazer um brinquedo funcionar!
